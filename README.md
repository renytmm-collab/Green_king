# 局域网双人格子建造与路线原型

当前版本提供六位房间、双人身份、服务器权威的 9×18 格子地图、金币、建筑和多段折线路线。左侧 6 列属于 player1，右侧 6 列属于 player2，中间 6 列为不可建造的中立战场。双方开局各有一座城堡，并可在自己的任意空格建造矿山、塔或兵营。

每位玩家以 400 金币开始；矿山 100、塔 150、兵营 180。每座矿山每 2 秒为所属玩家增加 10 金币。建筑位置、扣款、矿山收入和路线均由服务端验证并同步，客户端只负责输入与绘制。

## 启动与电脑/手机联测

首次运行先执行 `npm install`，之后执行 `npm start` 或双击 `start-server.cmd`。电脑浏览器打开终端中的本机地址并创建房间；将终端显示的 LAN 地址和六位房间号提供给同一 Wi-Fi 下的手机浏览器加入。Windows 防火墙提示时，只允许 Node.js 的专用网络通信。

双方会看到相同地图和建筑状态。点击自己领地的空格可选择矿山、塔或兵营。按住自己的兵营，用鼠标或手指在空地中绘制路线，并在敌方建筑上松开；合法路线同步到双方。同一兵营的新路线会覆盖旧路线，路线不能穿过其他建筑。Canvas 使用归一化逻辑坐标和 Pointer Events，支持窗口缩放、触摸以及高 DPI 屏幕。

运行自动测试：

```powershell
npm test
```

当前尚未实现士兵生成、移动、攻击、塔攻击、建筑生命、升级或胜负；美术仍是可替换的几何占位图形，不包含原游戏素材。
