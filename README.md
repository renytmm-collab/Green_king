# 局域网双人资源与建造原型

本阶段提供六位房间、双人身份、服务器权威的固定地图、金币、建筑和兵营路线。每位玩家以 400 金币开始；矿山 100、塔 150、兵营 180。每座矿山每 2 秒为所属玩家增加 10 金币。尚未实现士兵、移动、攻击、塔攻击、建筑生命、升级或胜负。

## 启动与联测

运行 `npm install`（首次）后执行 `npm start` 或双击 `start-server.cmd`。电脑创建房间，将终端显示的 LAN 地址和房间号给同一 Wi-Fi 的手机加入。

双方会看到相同建筑状态和自己的金币。点击自己的虚线空位，选择矿山、塔或兵营；服务器成功后扣款并同步。按住自己的初始或新建兵营拖到相邻节点可建立路线，矿山与塔不能建立路线。手机与电脑均使用 Pointer Events；Windows 防火墙提示时只允许 Node.js 的专用网络通信。
