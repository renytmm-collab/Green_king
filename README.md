# 局域网双人点击同步原型

本阶段只提供：启动局域网服务、创建/加入两人房间、左右身份分配，以及 Canvas 点击标记互相同步。没有矿山、塔、兵营、士兵、战斗或正式地图玩法。

## 安装与启动（电脑 A）

1. 安装 [Node.js LTS](https://nodejs.org/)。
2. 在项目目录运行 `npm install`。
3. 双击 `start-server.cmd`，或运行 `npm start`。
4. 终端会显示 `LAN address(es)`，例如 `http://192.168.1.23:3000`。

`start-server.cmd` 与服务都监听 `0.0.0.0`，因此可从局域网访问。

## 双机验收

1. 电脑 A 打开终端显示的 LAN 地址，点击“创建房间”，记下六位房间号；身份应为“左方（player1）”。
2. 电脑 B 接入同一 Wi-Fi/LAN，在浏览器打开同一个 LAN 地址，输入房间号并点击“加入房间”；身份应为“右方（player2）”。
3. 任一方点击画布：自己看到蓝色“我”，对方看到橙色“对方”。不同浏览器窗口大小不会改变位置对应关系。
4. 如 Windows Defender 防火墙弹窗出现，请允许 Node.js 在“专用网络”通信；或手动放行 TCP 3000 入站端口。不要自动修改防火墙。

## 常见问题与限制

- 在电脑 A 使用 `ipconfig`，找正在使用的网卡下的 **IPv4 地址**；不要将 `127.0.0.1` 发给电脑 B。
- 公共 Wi-Fi 可能启用了“客户端隔离”，此时两台设备无法互访；请改用不隔离的局域网或热点。
- 每个房间最多两人；关闭或刷新页面会离开房间，空房会自动删除。
- 本原型没有重连、聊天、存档、账号、数据库和公网部署。
