# Original visual assets (local-use build)

These PNG files were exported from the Flash version of *The Green Kingdom* for
the user's local, non-published build.

- Source SWF: `https://www.gamesflow.com/jeux/5559b4aa0bf7c.swf`
- Source SWF SHA-256: `89483C8ED13A96D61E9F46487955C59340C58D86E859DA407B2C5D47701B9B1F`
- Export tool: JPEXS Free Flash Decompiler 26.2.1
- Export method: named `DefineSprite` characters were rendered to transparent
  PNG frames; unit frames were packed without recoloring into 6 x 6 atlases.

Character mappings:

- `mc_Castle` -> castle
- `mc_Tower` -> tower
- `mc_Hut` -> barracks
- `mc_Market` -> mine
- `mc_Obstacles` frames 1 and 2 -> rock and tree
- `W1L` / `W1R` -> green soldier
- `W3L` / `W3R` -> purple soldier

The downloaded SWF, FFDec archive and bulk exports are intentionally not kept
in the project. Review the original game's rights before publishing or
redistributing these extracted images.
